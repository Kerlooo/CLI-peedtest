# Python CLI SpeedTest
# How to use it
## Debian
```bash
git clone https://github.com/Kerlooo/CLI-speedtest.git
cd CLI-speedtest
source myenv/bin/activate
pip install speedtest-cli
python3 speed_test.py
```

## Windows
```bash
git clone https://github.com/Kerlooo/CLI-speedtest.git
cd CLI-speedtest
pip install speedtest-cli
python3 speed_test.py
```
